# Installation

Use the repo root as the installation owner.

```bash
npm install
npm run dev
```

The root workspace installs `apps/web` and `packages/*` together. The root `package-lock.json` is the canonical lockfile.

## Current Requirements

- Node.js and npm for the website and package workspace.
- Python or Java tooling only when working on the corresponding research or desktop paths outside this website repo.

## Roadmap Requirements

C++, CUDA, TensorRT, broker SDKs, and live deployment infrastructure are not required for the current public web app. They belong to future execution/runtime lanes.
