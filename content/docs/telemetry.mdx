# Telemetry & metrics

This page describes observability for the current research substrate and planned future runtime surfaces. Current telemetry is about reproducible runs, artifact provenance, gate outcomes, and reviewable evidence. Production runtime telemetry is roadmap work.

---

## Current Scope

- Run identifiers and deterministic artifact references.
- Canonical bundle and provenance records.
- Validation outcomes, including leakage-aware split discipline.
- Gate status and reviewer-facing evidence summaries.

---

## Planned Runtime Telemetry

Future Phase II/III work may add latency, throughput, live position, execution, and model drift telemetry. Those values must not be published as product facts until they are measured from a shipped runtime with provenance.

Illustrative latency or throughput targets are roadmap targets only.
