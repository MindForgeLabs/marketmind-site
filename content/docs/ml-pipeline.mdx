# ML Pipeline

Planned Phase II work. No training, inference, meta-policy allocator, ONNX export, TensorRT optimization, or deployment pipeline is shipped as current product truth.

## Current Research Boundary

Phase II-0 is non-promotable scaffolding: artifact contract discipline and research-only pilot harness wiring. It is useful because it defines what evidence would be required. It is not allocator validation and does not move GATE-II.

## Proposed Candidate

The adaptive allocator is a future candidate. It may include context encoding, regime-indexed tasks, policy selection, and model export, but only if the validation scaffolding shows a viable path against a simpler incumbent baseline.

## Required Proof

Before this becomes product truth, MarketMind must prove:

- Leakage-safe training and validation discipline.
- Deterministic model and dataset artifacts.
- Canonical provenance for every promoted bundle.
- Statistical improvement over a simpler baseline.
- Governance approval through explicit gates.

Until that evidence exists, the allocator remains proposed research work.
